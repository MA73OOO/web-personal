# Contexto Específico para Gemini — Personal Web

Documento de directivas y contexto específico para el modelo **Gemini** (Antigravity IDE).

---

## 🎯 Enfoque de Trabajo de Gemini
1. **Calidad Visual:** Diseños modernos, limpios y atractivos con Tailwind CSS.
2. **Respeto al Usuario:** El usuario ejecuta directamente los comandos de terminal (`git`, `npm`, `npx`). Proporcionar siempre comandos Powershell claros y probados.
3. **Flujo de Trabajo:** Antes de sugerir commits, recordar verificar la salud del proyecto con los harneses (`npm run harness:git`).
4. **Bóveda:** Consultar y actualizar la carpeta `docs/` y `.agents/` ante cualquier cambio de arquitectura.

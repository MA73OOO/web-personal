# Personal Web — Showcase & Technical Hub

Bienvenido al repositorio de mi sitio web personal. Un proyecto construido paso a paso enfocado en diseño moderno, rendimiento extremo e infraestructura como código.

## 🌟 Características
* **Frontend:** React SPA impulsado por Vite.
* **Infraestructura:** AWS S3 + CloudFront CDN provisto con Terraform.
* **Gestión de Contenido:** Enfoque Jamstack Git-driven (archivos JSON/Markdown en repositorio).
* **Documentación:** Bóveda completa en `docs/` e instrucciones para agentes en `.agents/AGENTS.md`.
